# my-website

Personal GitHub Pages landing page for `vd4-dee`.

The page introduces `lam_data_nghiep_du`, highlights recent GitHub projects, and fetches the latest public repositories from the GitHub API with a local fallback snapshot.

## Local preview

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080`.

## Files

- `index.html`: static landing page with embedded CSS and JavaScript.
- `assets/developer-workspace-hero.jpg`: generated hero image for the portfolio.
